TURO-MOKO

A web-based e-learning platform designed to support advocacy-based and community-oriented education in Albay. Developed to address limited access to supplementary learning resources for Civil Society Organizations (CSOs) and Non-Government Organizations (NGOs), guided by the educational interventions of World Vision Development Foundation.

Key Features:

  • Interactive course delivery with resources and lessons
  • Assessment and evaluation system for learners
  • Course and program feedback monitoring deck
  • Role-based access control (Admin, Implementer, Learner)
  • Integrated chat system for user communication
  • Downloadable course materials and resources
